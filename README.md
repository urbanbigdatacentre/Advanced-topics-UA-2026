# Advanced Topics for Urban Analytics (2026)

Welcome to the repository for the "Advanced Topics for Urban Analytics" course. This repository contains two short excercises to visualise community-generated data

## Excercise 1: Visualise geojson data from the Sketchmap tool
In this excercise, we will visualise a geojson file using well known libraries and the results from the digitalisation of the map created by the students in class. The files required must be generated after drawing on the paper map and processing it using [the Sketchmap tool](https://sketchmaptool.org).

The excercise is available in the [a_smt_geojson_overlay.ipynb](a_smt_geojson_overlay.ipynb) notebook.

## Excercise 2: Visualise community-generated data on top of an open source dataset
In this excercise, we will visualise the data generated by the students in class using the Sketchmap tool and a Kobo Toolbox Survey. We use the map of urban deprivation provided by the city of Glasgow as the open source dataset to visualise the data on top of it.

The excercise is available in the [b_urban_deprivation_overlay.ipynb](b_urban_deprivation_overlay.ipynb) notebook.
